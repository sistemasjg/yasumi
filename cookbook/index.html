Yasumi Recipes

Interested in learning more about Yasumi? This section includes various recipes, examples and how-to's to get started with Yasumi. For a basic primer, please check out the Getting Started section.

A quick reminder: here's how easy it is to start using Yasumi in your project:

require 'vendor/autoload.php';

// Use the Yasumi factory to create a new holiday provider instance
$holidays = Yasumi\Yasumi::create('USA', 2017);
